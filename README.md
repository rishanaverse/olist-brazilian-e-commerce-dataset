# 📊olist-brazilian-e-commerce-dataset


📌 Project Overview

This project presents an interactive Power BI dashboard built using the Olist Brazilian E-Commerce dataset. The goal of this analysis is to transform raw e-commerce data into meaningful business insights by exploring sales performance, customer behavior, product trends, delivery efficiency, and marketplace operations.

The dashboard helps visualize key performance indicators (KPIs) and identify patterns that can support data-driven decision-making.

⸻

📂 Dataset

The project uses the Olist Brazilian E-Commerce dataset, a real-world dataset containing information about orders placed on a Brazilian online marketplace.

The dataset includes:

* 🛒 Customer details
* 📦 Order information
* 🏷️ Product details
* 💳 Payment transactions
* ⭐ Customer reviews
* 🚚 Delivery information
* 🏪 Seller information

⸻

🛠️ Tools & Technologies Used

* Microsoft Power BI – Data visualization and dashboard development
* Power Query – Data cleaning and transformation
* DAX – Creating calculated measures and KPIs
* Data Modeling – Building relationships between multiple tables

⸻

The Power BI dashboard includes analysis of:
